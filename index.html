"Breakout"

Game designed by Dennis Comella

Code your own with this nifty tutorial: 2D breakout game using pure JavaScript


Controls:

  • P = Pause
  • M = disable/enable Mouse controls
  • Left < and Right > arrow keys to move
  • prrr = Restart game & score
  • pddd + {e, h, u, l} = change Difficulty settings (re-randomizes board)
  • F / S = make the game Faster or Slower
